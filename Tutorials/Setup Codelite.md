# Setup Codelite

Reasons we recommend Codelite:
* It's small
* Low resource usage
* Easy to install
* Code folding
* Workspaces so you can work on multiple projects
* Multi-editing

Get Codelite for Ubuntu from here
https://docs.codelite.org/downloads/download/#ubuntu-debian

## Setup External tools
* mudeer-build
* mudeer-install
* mudeer-release
