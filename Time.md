# Time

## Use Case

When you have unix timestamps & want to display them dynamically in many formats.

```js
Time.now();
Time.format( Time.now(), 'YYYY MM DD' );
```

```js
[id=time] [by=year]
[id=time] [by=month]
[id=time] [by=day]
[id=time] [by=age]
```
